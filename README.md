# @hbs/cli

### 介绍
> This is Hongbusi's custom cli.

### 常用命名

- 查看版本
```
hbs -V | --version
```

- 查看所有模板
```
hbs list
```

- 使用模板创建项目
```
hbs init <templateName> <projectName>
```

### 已有模板
- vue-template

### 解决方案
- 如一直卡在 Downloading template... ?
```
// 切换 npm 源
npm config set registry https://registry.npm.taobao.org
```
